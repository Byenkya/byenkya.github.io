Intro

Market Assistant Web Service (MWAS)

This project was designed to help Ugandan farmers expand thier customer bases and profit margins.

View Application

Features

Register User Feature

The register page allows one to join the system by providing all the required personal and demographic information. The user should provide the following information:

  • First name (required)
  • Last Name (Required)
  • Gender (Required)
  • Email (Optional)
  • Telephone(Required)
  • Address (Required)
  • Username (Required)
  • Password (Required)
After signup the user must verify his/her account created.

Login Feature

The login page allows a registered person to have access to the system resources through navigations. For a user to be validated and authenticated to use the system he/she must provide the following information.

  • Username (Required)
  • Password (Required)

DashBoard Feature

The dashboard will contain the different activities a farmer can carry out using the web service portal for example registering a product, viewing available customers for his agricultural produce, complaint and advertising.

About

Market Assistant Web Service Portal

Timeline

6 months

I found out that, the government of Uganda seeks to undertake improvements in infrastructure such as good roads to link farm premises and markets, agro-product processing factories like maize millers factory in Kawanda and storage facilities to store large quantities of farm outputs like maize stores and rice stores("Uganda: Community Agricultural Infrastructure Improvement Programme", 2006). However, as the government tries to do all its best in improving the agricultural sector, farmers still have limited access to markets especially local markets which includes processes of identifying buyers of the farm outputs, finding prices as well as negotiating prices. This is due to lack of market information which inhibits the sale of their produce to the markets hence lowering farmers and domestic income (Payne, 2002). In response to the above problem, I proposed to design and implement a web service to help farmers realize greater sales of their agricultural produce by enhancing communication between farmers and customers, thus improving on their income and increase the countryâ€™s GNP.
